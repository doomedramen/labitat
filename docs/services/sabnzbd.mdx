# SABnzbd

Usenet downloader.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | SABnzbd server URL |
| API Key | Password | Yes | SABnzbd API key |

## Finding Your API Key

1. Open SABnzbd
2. Go to **Config → General**
3. Find **Security → API Key**

## Widget Displays

- Queue size
- Download speed
- Disk space
