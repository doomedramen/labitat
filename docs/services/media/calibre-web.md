# Calibre-Web

Web-based e-book library manager for your Calibre collection.

## Configuration

| Field | Type | Required | Description                                              |
| ----- | ---- | -------- | -------------------------------------------------------- |
| URL   | URL  | Yes      | Calibre-Web URL (e.g., `https://calibre-web.local:8083`) |

## Widget Displays

- Total books count
- Total authors count
- Total categories count
- Total series count

## Notes

This widget displays statistics from your Calibre-Web library by querying the OPDS stats endpoint. It shows the total number of books, authors, categories, and series in your collection.
