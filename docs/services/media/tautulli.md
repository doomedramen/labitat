# Tautulli

Plex monitoring and statistics.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | Tautulli server URL |
| API Key | Password | Yes | Tautulli API key |

## Finding Your API Key

1. Open Tautulli
2. Go to **Settings → Web Interface**
3. Enable API and copy the key

## Widget Displays

- Active streams
- Recent activity
- Server stats
