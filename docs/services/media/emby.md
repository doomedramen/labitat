# Emby

Media server.

## Configuration

| Field   | Type     | Required | Description     |
| ------- | -------- | -------- | --------------- |
| URL     | URL      | Yes      | Emby server URL |
| API Key | Password | Yes      | Emby API key    |

## Finding Your API Key

1. Open Emby Dashboard
2. Go to **Advanced → API Keys**
3. Create a new API key

## Widget Displays

- Active streams
- Server status
- Library stats
