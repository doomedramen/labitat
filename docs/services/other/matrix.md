# Matrix Rain

Animated Matrix digital rain screensaver effect.

## Configuration

This widget requires no configuration. Simply add it to your dashboard to display the animation.

## Widget Displays

- Animated falling characters from the Matrix movie
- Katakana characters with green trail effect
- Bright white head characters
- Smooth animation with varying speed

## Notes

This is a purely decorative, client-side widget that creates an animated Matrix-style digital rain effect. It uses HTML5 Canvas for rendering and requires no server connection or API keys. The animation features:

- Random character trails with varying lengths
- Fade effect from bright green to dark
- Staggered column entry for natural appearance
- Responsive to container size
