# Pipes

Animated pipes screensaver effect.

## Configuration

This widget requires no configuration. Simply add it to your dashboard to display the animation.

## Widget Displays

- Animated colored pipes moving across the screen
- Multiple simultaneous pipe animations
- Smooth fade-out trail effect
- Junction dots at pipe corners

## Notes

This is a purely decorative, client-side widget that creates an animated pipes screensaver-style effect. It uses HTML5 Canvas for rendering and requires no server connection or API keys. The animation features:

- Multiple colored pipes (3-7 simultaneous)
- 8 different pipe colors
- Random turns at junctions (no 180° turns)
- Smooth fade-out trail effect
- Edge wrapping for continuous animation
- Responsive to container size
