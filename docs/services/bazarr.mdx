# Bazarr

Subtitle manager for Sonarr and Radarr.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | Bazarr server URL |
| API Key | Password | Yes | Bazarr API key |

## Finding Your API Key

1. Open Bazarr
2. Go to **Settings → Security**
3. Copy the API key

## Widget Displays

- Wanted subtitles count
- Missing subtitles
- History
