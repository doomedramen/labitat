# Glances

Cross-platform system monitoring.

## Configuration

| Field    | Type     | Required | Description      |
| -------- | -------- | -------- | ---------------- |
| URL      | URL      | Yes      | Glances API URL  |
| Username | Text     | No       | Glances username |
| Password | Password | No       | Glances password |

## Widget Displays

- CPU usage
- Memory usage
- Disk usage
- Network I/O
