# Portainer

Container management UI.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | Portainer server URL |
| API Key | Password | Yes | Portainer API key |

## Finding Your API Key

1. Open Portainer
2. Go to **User → API Keys**
3. Create a new API key

## Widget Displays

- Running containers
- Stack count
- Node count
