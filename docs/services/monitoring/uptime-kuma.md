# Uptime Kuma

Uptime monitoring.

## Configuration

| Field               | Type     | Required | Description            |
| ------------------- | -------- | -------- | ---------------------- |
| URL                 | URL      | Yes      | Uptime Kuma server URL |
| API Key (MFA Token) | Password | No       | MFA token if enabled   |

## Widget Displays

- Up/down count
- Recent incidents
- Overall uptime
