# Glances Processes

Top resource-consuming processes from Glances.

## Configuration

| Field         | Type     | Required | Description                                  |
| ------------- | -------- | -------- | -------------------------------------------- |
| URL           | URL      | Yes      | Glances API URL                              |
| Username      | Text     | No       | Glances username                             |
| Password      | Password | No       | Glances password                             |
| Process count | Select   | No       | Number of processes to display (5, 8, or 10) |
| Sort by       | Select   | No       | Sort by CPU usage or Memory usage            |

## Widget Displays

- Top processes by CPU or memory usage
- Process name
- CPU usage percentage
- Memory usage percentage
- Relative usage bars with color coding

## Notes

Displays the most resource-intensive processes running on your system. You can choose to sort by either CPU or memory usage and control how many processes are shown.
