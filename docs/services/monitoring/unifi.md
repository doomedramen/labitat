# Unifi

Network controller.

## Configuration

| Field    | Type     | Required | Description          |
| -------- | -------- | -------- | -------------------- |
| URL      | URL      | Yes      | Unifi controller URL |
| Username | Text     | Yes      | Unifi username       |
| Password | Password | Yes      | Unifi password       |

## Widget Displays

- Connected clients
- WAN status
- Alerts
