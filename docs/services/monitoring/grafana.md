# Grafana

Metrics dashboard and visualization.

## Configuration

| Field   | Type     | Required | Description                   |
| ------- | -------- | -------- | ----------------------------- |
| URL     | URL      | Yes      | Grafana server URL            |
| API Key | Password | Yes      | Grafana service account token |

## Finding Your API Key

1. Open Grafana
2. Go to **Administration → API Keys**
3. Create a new key

## Widget Displays

- Dashboard count
- Alert count
- Datasource count
