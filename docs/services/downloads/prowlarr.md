# Prowlarr

Indexer manager for \*arr applications.

## Configuration

| Field   | Type     | Required | Description         |
| ------- | -------- | -------- | ------------------- |
| URL     | URL      | Yes      | Prowlarr server URL |
| API Key | Password | Yes      | Prowlarr API key    |

## Finding Your API Key

1. Open Prowlarr
2. Go to **Settings → General**
3. Scroll to **API Key**
4. Copy the key

## Widget Displays

- Indexer count
- Grab count
- Failed grab count
