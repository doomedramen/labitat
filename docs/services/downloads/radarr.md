# Radarr

Movie collection manager.

## Configuration

| Field   | Type     | Required | Description       |
| ------- | -------- | -------- | ----------------- |
| URL     | URL      | Yes      | Radarr server URL |
| API Key | Password | Yes      | Radarr API key    |

## Finding Your API Key

1. Open Radarr
2. Go to **Settings → General**
3. Scroll to **API Key**
4. Copy the key

## Widget Displays

- Queue count
- Disk space usage
- Missing movies
