# Jackett

API proxy for torrent trackers.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | Jackett server URL |
| API Key | Password | Yes | Jackett API key |

## Finding Your API Key

1. Open Jackett
2. Copy the API key from the top-right corner

## Widget Displays

- Configured indexers count
- Status
