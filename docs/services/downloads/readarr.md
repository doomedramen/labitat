# Readarr

Book and ebook collection manager.

## Configuration

| Field   | Type     | Required | Description        |
| ------- | -------- | -------- | ------------------ |
| URL     | URL      | Yes      | Readarr server URL |
| API Key | Password | Yes      | Readarr API key    |

## Finding Your API Key

1. Open Readarr
2. Go to **Settings → General**
3. Scroll to **API Key**
4. Copy the key

## Widget Displays

- Queue count
- Disk space usage
- Missing books
