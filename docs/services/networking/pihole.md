# Pi-hole

Network-wide ad and tracker blocking.

## Configuration

| Field              | Type     | Required | Description          |
| ------------------ | -------- | -------- | -------------------- |
| URL                | URL      | Yes      | Pi-hole admin URL    |
| API Key (Password) | Password | Yes      | Pi-hole web password |

## Widget Displays

- Queries blocked
- Total queries
- Domains on blocklist
