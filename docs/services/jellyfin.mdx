# Jellyfin

Open-source media server.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | Jellyfin server URL |
| API Key | Password | Yes | Jellyfin API key |

## Finding Your API Key

1. Open Jellyfin
2. Go to **Dashboard → API Keys**
3. Create a new API key

## Widget Displays

- Active streams
- Server status
- Library stats
