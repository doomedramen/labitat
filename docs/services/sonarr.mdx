# Sonarr

TV series collection manager.

## Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| URL | URL | Yes | Sonarr server URL |
| API Key | Password | Yes | Sonarr API key |

## Finding Your API Key

1. Open Sonarr
2. Go to **Settings → General**
3. Scroll to **API Key**
4. Copy the key

## Widget Displays

- Queue count
- Disk space usage
- Missing episodes
